Alien Facility Escape

Version 1.0 • 2026

Alien Facility Escape is a sci-fi interactive web adventure game. You awaken inside an alien containment facility, forced to make choices that determine survival, sacrifice, or selfish escape.

Game Features

  • Branching narrative paths
  • Multiple endings
  • Inventory-based decisions
  • Dynamic background transitions
  • Health system with consequences
  • Moral choice endings

Technology Used

  • HTML5
  • CSS3 (Custom UI Design)
  • Vanilla JavaScript
  • LocalStorage Save System
